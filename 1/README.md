# 1.02 プログラムの書き方とエラー

- エラーの種類
  - コンパイルエラー ... 文法のエラーで、プログラムは実行されない(ex: cout < "test" << endl;)
  - 実行時エラー ... 内容のエラーで、プログラムは強制終了される(ex: 3/0)
  - 論理エラー ... 内容のエラーで、プログラムは正しく動いているように見えてしまう(ex: expected: 300+100 but 300-100)
